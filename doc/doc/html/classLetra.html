Detailed Description

T.D.A Letras.

Definición: Una instancia l del tipo de dato abstracto Letras se puede construir como

  • Un char y dos int inicializados a un serie de valores dados
  • Como una letra "vacía" en la que los elementos tomarán valores por defecto. Cada Letra representa un carácter junto sus apariciones y puntuación.

Para poder usar el TDA Letra se debe incluir el fichero

#include Letras.h
