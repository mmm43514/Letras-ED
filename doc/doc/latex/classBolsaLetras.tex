\subsection{Detailed Description}
T.\-D.\-A \hyperlink{classBolsaLetras}{Bolsa\-Letras}. 

{\bfseries Definición\-:} Una instancia {\itshape a} del tipo de dato abstracto \hyperlink{classBolsaLetras}{Bolsa\-Letras} se construye como un objeto vacío que se rellena a través de un \hyperlink{classConjuntoLetras}{Conjunto\-Letras}

La \hyperlink{classBolsaLetras}{Bolsa\-Letras} se monta en base y a través del T.\-D.\-A \hyperlink{classLetra}{Letra} y el T.\-D.\-A \hyperlink{classConjuntoLetras}{Conjunto\-Letras}.

Almacena tantas veces repetidas las letras de \hyperlink{classConjuntoLetras}{Conjunto\-Letras} como indique su campo apariciones.
